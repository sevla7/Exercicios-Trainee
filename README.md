# Exercícios Trainee

Repositório contendo a resolução de exercícios, desafios e projetos desenvolvidos durante o projeto **Trainee** da CATI Jr, com foco no desenvolvimento de lógica de programação, análise de dados, estatística, machine learning e boas práticas de desenvolvimento.

Ao longo das semanas foram aplicados conceitos fundamentais da Ciência de Dados por meio de atividades práticas, projetos completos e estudos utilizando Python e suas principais bibliotecas.

---

## Objetivos

Este repositório foi criado para:

- Desenvolver lógica de programação e resolução de problemas;
- Consolidar conhecimentos adquiridos durante o programa Trainee;
- Praticar análise exploratória e manipulação de dados;
- Aplicar técnicas estatísticas para análise de dados;
- Desenvolver modelos de Machine Learning supervisionado e não supervisionado;
- Registrar a evolução do aprendizado através de exercícios e projetos práticos.

Repositório contendo a resolução de exercícios e desafios propostos durante o programa Trainee, com foco no desenvolvimento de lógica de programação, estruturas de dados, algoritmos e boas práticas de desenvolvimento.

## Objetivo

Este repositório foi criado para:

- Praticar conceitos fundamentais de Python;
- Desenvolver raciocínio lógico e resolução de problemas;
- Consolidar conhecimentos adquiridos durante o programa de treinamento;
- Registrar a evolução do aprendizado através de exercícios práticos.

## Tecnologias Utilizadas

- Python
- Git e GitHub

- ## Conceitos Praticados

- Variáveis e tipos de dados
- Estruturas condicionais
- Estruturas de repetição
- Funções e métodos
- Vetores e matrizes
- Manipulação de arquivos
- Numpy e Pandas
- Estruturas de dados
- Boas práticas de programação

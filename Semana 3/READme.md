# 📊 **Análise Estatística sobre o Impacto das Redes Sociais na Saúde Mental dos Estudantes**

## 📖 Sobre o Dataset

Este dataset reúne **5.000 registros de estudantes** e foi desenvolvido para investigar a relação entre:

- 📱 Uso de redes sociais;
- 🌱 Hábitos de estilo de vida;
- 🧠 Bem-estar e saúde mental.

O objetivo é possibilitar análises que ajudem a compreender como o comportamento digital pode influenciar a qualidade de vida e a saúde mental dos estudantes.

## 🎯 Potenciais Aplicações

Este conjunto de dados pode ser utilizado para diversas análises, como:

- 📈 Tendências de uso das redes sociais entre estudantes;
- ⭐ Análise da popularidade das diferentes plataformas;
- ⏱️ Relação entre tempo de uso e indicadores de saúde mental;
- 😴 Impacto dos hábitos de vida (como sono e atividade física) no bem-estar;
- 📊 Desenvolvimento de modelos estatísticos e preditivos relacionados à saúde mental.
